Установка и настройка XRay на клиенте

1. Устанавливаем nekoray

Скачиваем nekoray с сайта На системах, основанных на Debian, скачиваем deb-пакет и устанавливаем.

На системах, основанных на Arch Linux, проще всего добавить репозиторий chaotic-aur и установить Nekoray оттуда. Также нужно будет установить библиотеки sing-geosite и sing-geoip:

sudo pacman -S nekoray sing-geosite sing-geoip

Однако в дистрибутиве Manjaro Nekoray, установленный таким способом, возможно, не запустится из-за проблем с зависимостями. При этом, например, в дистрибутиве EndeavourOS Nekoray, установленный из chaotic-aur, успешно запускается и работает.

Если Nekoray, установленный таким способом, не работает либо у нас другой дистрибутив Linux, можно просто скачать архив для Linux с этого же сайта и запустить. Там же есть архив и для Windows.

Версию AppImage не стоит скачивать, если мы хотим использовать VPN для всей системы: она не поддерживает режим TUN.

2. Настраиваем подключения

Идем в Сервер->Новый профиль, и далее заполняем поля следующим образом.

2.1 Для ShadowSocks

Поле Значение
Тип ShadowSocks
Имя Название профиля
Адрес наш_домен
Порт порт ShadowSocks
Шифрование 2022-blake3-aes-128-gcm или 2022-blake3-aes-256-gcm
Пароль наш_пароль

2.2 Для прямого VLESS + XTLS-Vision

Поле Значение
Тип VLESS
Имя Название профиля
Адрес наш_домен
Порт 443
UUID наш_uuid
Flow xtls-rprx-vision
Транспорт tcp
Безопасность tls
Кодирование пакетов xudp
Fingerprint chrome

2.3 Для VLESS-over-Websockets

Поле Значение
Тип VLESS
Имя Название профиля
Адрес наш_домен
Порт 443
UUID наш_uuid
Транспорт ws
Безопасность tls
Кодирование пакетов xudp
Path /myverysecretpath
Fingerprint chrome

3. Дополнительные настройки

Можно выбрать порт для входящих подключений - тот порт, на котором у нас будет висеть XRay на клиенте, по умолчанию 2080.

4. Тест и запуск

Выбираем нужное подключение в списке на главном окне, кликаем правой кнопкой мыши -> Текущий выбор -> URL test, и видим в логе и в окошке, что пинг успешен.

Если это так, то можем запустить выбранный профиль, кликнув правой кнопкой мыши -> Запустить. Или просто нажав клавишу Enter.
